To run the Weather App project, you'll need to follow these steps:
First you need to clone this repo after cloning follow the velow steps
1. Backend Setup:
   - Ensure you have Python installed on your system.
   - Install Flask and other required Python packages using pip. You can do this by navigating to the backend directory in your terminal and running:
     ```
     pip install -r requirements.txt
     ```
   - Make sure you have PostgreSQL installed and running on your system. You'll also need to create a PostgreSQL database named `weather-app`.

2. Frontend Setup:
   - Make sure you have Node.js and npm installed on your system.
   - Navigate to the frontend directory in your terminal and run:
     ```
     npm install
     ```

3. Running the Backend:
   - Inside the backend directory, run the following command in your terminal:
     ```
     python app.py
     ```

4. Running the Frontend:
   - Inside the frontend directory, run the following command in your terminal:
     ```
     npm start
     ```

5. Accessing the App:
   - Once both the backend and frontend are running, you can access the Weather App by opening your web browser and navigating to `http://localhost:3000`.

These steps will install all necessary packages and start both the frontend and backend servers, allowing you to access and use the Weather App.
